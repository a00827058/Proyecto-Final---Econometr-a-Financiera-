# Proyecto Final Econometría Financiera 
Este proyecto desarrolla un análisis econométrico completo de un portafolio de 12 activos financieros seleccionados en HowMarketsWorks, utilizando datos intradiarios de 30 minutos descargados de Refinitiv/Eikon para el periodo del 10 de enero de 2023 al 16 de marzo de 2026. Todo el análisis fue desarrollado en R con RStudio sobre Posit Cloud.
